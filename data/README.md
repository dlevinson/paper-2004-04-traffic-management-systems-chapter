# Data

Updated: 2026-05-17 03:07:53 AEST

This folder stages compact Stata regression inputs for the traffic-management-systems chapter. The data match the chapter's stated analysis of 1998-2000 Twin Cities freeway segment speed, density, ramp metering, variable-message-sign, Highway Helper, HOV, and incident variables.

Included groups:

- `original/stata/primary_regression_inputs`: four time-window DTA files used for the main speed regressions.
- `original/stata/incident_free_subsets`: four incident-free DTA subsets.
- `original/stata/incident_only_subsets`: four incident-only DTA subsets.

Large upstream route spreadsheets and intermediate Excel workbooks are not copied into this compact package. Their source locations are documented in `metadata/SOURCE_BOUNDARY_DECISIONS.csv`.
